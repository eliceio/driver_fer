## Requirements
Initial requirements are as follows.
```
  Python 3.5
  tensorflow 1.10.0
  opencv-python 3.4.2.17
  pandas 0.23.4
  keras 2.2.2
  imutils 0.5.0
  dlib 19.1.0
```
Installation: 
```
pip install -r requirements.txt 
```
